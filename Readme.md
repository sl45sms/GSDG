Greek Synthetic Data Generation

This repository contains code for generating synthetic data in Greek from glossApi corpus. 

The code is designed to create realistic and diverse datasets at  Conversational Format (ChatML), which can be used for various natural language processing tasks.

The model used for generating synthetic data is the latest Qwen3-235B-A22B, which is a powerful language model capable of producing high-quality text in Greek, better even than larger models like Deepseek3-671B.

The synthetic data generated by this code can be used to fine-tune language models like Swich AI - Apertus, train machine learning algorithms, and conduct research in the field of natural language processing.
